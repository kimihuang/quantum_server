    Zephyr Hello World 编译与运行指南¶

    环境概述¶

    项目 值
    Zephyr 版本 v4.4.99 (main 分支)
    West 版本 v1.5.0
    Zephyr SDK 1.0.1
    Python 虚拟环境 /home/lion/workdir/sourcecode/quantum_main/.venv
    SDK 路径 /home/lion/.local/zephyr-sdk-1.0.1
    Workspace 路径 /home/lion/workdir/sourcecode/quantum_main/src/zephyrproject

    目录结构¶

    zephyrproject/          # west workspace 根目录
  .west/                # west 配置
  zephyr/               # zephyr manifest 仓库
  modules/              # west 管理的外部模块
  bootloader/           # bootloader 模块
  tools/                # 工具链相关
  build/                # 编译输出目录（west build 生成）

    前置条件¶

    1. 系统依赖¶

    # 需要安装的工具
sudo apt install cmake ninja-build dtc qemu-system-arm

    2. Python 依赖¶

    使用项目 venv 安装：

    # 激活虚拟环境
source /home/lion/workdir/sourcecode/quantum_main/.venv/bin/activate

# 安装 west
pip install west

# 安装 zephyr 编译所需 Python 包
pip install kconfiglib pyelftools cmake ninja jsonschema

    3. Zephyr SDK¶

    从 https://github.com/zephyrproject-rtos/sdk-ng/releases 下载安装，本环境安装于：

    /home/lion/.local/zephyr-sdk-1.0.1

    4. 环境变量¶

    # 设置 SDK 路径（必须，否则编译找不到工具链）
export ZEPHYR_SDK_INSTALL_DIR=/home/lion/.local/zephyr-sdk-1.0.1

    建议将此变量加入 ~/.bashrc 中。

    初始化 Workspace¶

    方式一：从已有 zephyr 仓库初始化（推荐）¶

    适用于已经通过 git clone 获取了 zephyr 代码的情况：

    # 将已有的 zephyr 仓库放入 workspace 目录
mkdir -p /path/to/zephyrproject
mv /path/to/zephyr /path/to/zephyrproject/zephyr

# 进入 workspace 目录
cd /path/to/zephyrproject

# 用本地仓库初始化 west
west init -l zephyr

# 拉取所有模块
west update

    方式二：从远程仓库初始化¶

    west init /path/to/zephyrproject
cd /path/to/zephyrproject
west update

    编译 Hello World¶

    # 进入 workspace 目录
cd /home/lion/workdir/sourcecode/quantum_main/src/zephyrproject

# 设置环境变量
export ZEPHYR_SDK_INSTALL_DIR=/home/lion/.local/zephyr-sdk-1.0.1

# 编译（-p auto 表示每次构建前自动清理）
west build -p auto -b qemu_cortex_m3 zephyr/samples/hello_world

    编译输出摘要：

    Memory region         Used Size  Region Size  %age Used
           FLASH:        9472 B       256 KB      3.61%
             RAM:        4104 B        64 KB      6.26%
        IDT_LIST:           0 B        32 KB      0.00%

    运行 Hello World¶

    # 在 QEMU 中运行（需要 qemu-system-arm）
west build -t run

    退出 QEMU：按 Ctrl+a，然后按 x。

    预期输出：

    *** Booting Zephyr OS build v4.4.0-1477-g1f77bdd7032d ***
Hello World! qemu_cortex_m3/ti_lm3s6965

    常见问题¶

    pip 安装失败（SOCKS 代理）¶

    如果系统设置了 socks5:// 代理，pip 会因缺少 socks 支持而报错：

    ERROR: Could not install packages due to an OSError: Missing dependencies for SOCKS support.

    解决方法：临时清除代理环境变量：

    HTTPS_PROXY="" HTTP_PROXY="" pip install <package>

    west init 克隆失败（pack 文件损坏）¶

    网络不稳定时 west init 克隆 manifest 仓库容易失败，报 invalid index-pack output。

    解决方法：先手动 git clone zephyr 仓库，再用 west init -l 从本地初始化。

    缺少 jsonschema¶

    编译时报 Missing jsonschema dependency，安装即可：

    pip install jsonschema

    常用 Samples 编译与运行¶

    以下示例均可在 qemu_cortex_m3 目标板上编译运行。

    编译命令通用格式¶

    # 使用独立 build 目录避免冲突（-d 指定输出目录）
west build -p -b qemu_cortex_m3 -d build_xxx zephyr/samples/<path>

# 运行
west build -d build_xxx -t run

    示例一览¶

    示例 路径 说明 FLASH RAM
    hello_world samples/hello_world 基础入门，打印 Hello World 9472 B 4104 B
    minimal samples/basic/minimal 最小化空应用 9368 B 4104 B
    philosophers samples/philosophers 经典哲学家就餐，多线程同步 13224 B 17672 B
    synchronization samples/synchronization 线程同步示例 10952 B 6568 B
    condition_variables samples/kernel/condition_variables/condvar 条件变量使用 21092 B 7752 B
    msg_queue samples/kernel/msg_queue 消息队列（含紧急消息） 11032 B 8474 B
    shell samples/subsys/shell/shell_module Shell 命令行交互 60000 B 13064 B

    hello_world¶

    west build -p -b qemu_cortex_m3 -d build_hello zephyr/samples/hello_world
west build -d build_hello -t run

    输出：

    *** Booting Zephyr OS build v4.4.0-1477-g1f77bdd7032d ***
Hello World! qemu_cortex_m3/ti_lm3s6965

    minimal¶

    最小化应用，启动后无额外输出。用于验证工具链和编译流程是否正常。

    west build -p -b qemu_cortex_m3 -d build_minimal zephyr/samples/basic/minimal
west build -d build_minimal -t run

    philosophers¶

    演示哲学家就餐问题，展示多优先级抢占式/协作式线程、互斥锁和线程睡眠。

    west build -p -b qemu_cortex_m3 -d build_philosophers zephyr/samples/philosophers
west build -d build_philosophers -t run

    输出：

    Philosopher 4 [C:-1]        STARVING
Philosopher 4 [C:-1]    HOLDING ONE FORK
Philosopher 4 [C:-1]   EATING  [  25 ms ]
Philosopher 5 [C:-2]        STARVING
...

    synchronization¶

    演示多线程同步，两个线程交替打印消息。

    west build -p -b qemu_cortex_m3 -d build_sync zephyr/samples/synchronization
west build -d build_sync -t run

    输出：

    thread_a: Hello World from cpu 0 on qemu_cortex_m3!
thread_b: Hello World from cpu 0 on qemu_cortex_m3!
...

    condition_variables¶

    演示条件变量的使用。消费者线程等待计数器达到阈值，生产者线程递增计数器。

    west build -p -b qemu_cortex_m3 -d build_condvar zephyr/samples/kernel/condition_variables/condvar
west build -d build_condvar -t run

    输出：

    watch_count: thread 1 Count= 0. Going into wait...
inc_count: thread 2, count = 1, unlocking mutex
inc_count: thread 3, count = 2, unlocking mutex
...
inc_count: thread 3, count = 12  Threshold reached.Just sent signal.
watch_count: thread 1 Condition signal received. Count= 12

    msg_queue¶

    演示消息队列的使用，包含普通消息和紧急消息（urgent message）。

    west build -p -b qemu_cortex_m3 -d build_msgq zephyr/samples/kernel/msg_queue
west build -d build_msgq -t run

    输出：

    [producer] sending: 0
[producer] sending: A (urgent)
[consumer] got sequence: CBA012345

    shell¶

    Zephyr 内置 Shell 命令行，提供设备管理、日志查看等交互命令。

    west build -p -b qemu_cortex_m3 -d build_shell zephyr/samples/subsys/shell/shell_module
west build -d build_shell -t run

    Shell 需要交互式终端输入命令。退出 QEMU：Ctrl+a 再按 x。

    不适用于 qemu_cortex_m3 的示例¶

    以下示例需要特定硬件（LED、GPIO 等），无法在 QEMU 上运行：

    • samples/basic/blinky — 需要 LED 设备节点
    • samples/basic/threads — 需要 LED 设备节点
    • samples/basic/button — 需要按键
    • samples/drivers/* — 需要对应硬件外设
