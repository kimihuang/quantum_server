CMN-600 模块公共接口定义

定义 CMN-600 (Arm Coherent Mesh Network) 互连控制器模块的 公共接口，包括模块 API 索引、内存区域类型、配置数据结构、 CCIX 配置接口和 PPU 观察者 API。外部模块通过这些接口 配置和查询 CMN-600 互连拓扑。

注解
该模块运行在 SCP (System Control Processor) 固件框架中， 遵循 SCP 模块的 init/bind/start 生命周期。
