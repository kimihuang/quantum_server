详细描述

CMN-600 互连控制器工具函数实现

实现 CMN-600 互连控制器的节点遍历、查询、地址编码 和根节点定位等工具函数。这些函数被模块主实现 (mod_cmn600.c) 和 CCIX 实现 (cmn600_ccix.c) 共同使用。

根据外设基地址、HN-D 节点 ID 和 mesh 尺寸获取根节点描述符

根据 mesh 尺寸确定坐标编码位数，从 HN-D 节点 ID 中 提取坐标信息，计算根节点（Configuration Manager）的地址偏移。 该函数同时设置全局变量 encoding_bits 和 mask_bits， 供 get_node_pos_x() 和 get_node_pos_y() 使用。
