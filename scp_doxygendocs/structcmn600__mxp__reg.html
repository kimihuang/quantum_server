详细描述

XP（交叉点）寄存器

XP 是 CMN-600 mesh 中的路由节点，连接 mesh 中相邻的节点。 每个 XP 有两个端口（port 0 和 port 1），可连接到各类节点。 XP 包含端口连接信息、QoS 配置、系统一致性请求控制、 调试跟踪模块 (DTM) 和错误报告寄存器。
