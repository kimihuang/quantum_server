详细描述

CCIX Gateway Link Agent (CXLA) 寄存器映射

CXLA 是 CCIX 网关中的链路代理，负责 CCIX 链路的协议协商和配置， 包括消息打包、最大包大小、PCIe TLP 头字段等。 每个 CXLA 对应一个物理 CCIX 链路。

FWK_RW uint64_t cmn600_cxla_reg::CXLA_AGENTID_TO_LINKID_REG[8]

AgentID 到 LinkID 映射寄存器数组

每个 64 位寄存器容纳 8 个 AgentID-to-LinkID 映射项（每项 8 位）， 共 8 个寄存器可映射 64 个代理。
