详细描述

CMN-600 模块内部上下文结构定义

定义 CMN-600 驱动模块的运行时上下文结构体 cmn600_ctx， 保存互连拓扑发现结果、节点配置信息和 CCIX 相关状态。 该头文件仅供模块内部使用，不属于公共接口。

注解
该驱动运行在 SCP (System Control Processor) 固件环境中， 使用 SCP 框架的内存管理和定时器 API。

◆ cmn600_setup_sam()

int cmn600_setup_sam ( struct cmn600_rnsam_reg *  rnsam)

配置 RN-SAM（请求节点系统地址映射）

根据模块配置中的内存区域映射表，对指定的 RN-SAM 节点 进行地址映射配置，包括 IO 区域、系统缓存区域和 CCIX 区域。

参数
[in]rnsam指向待配置的 RN-SAM 寄存器结构体
返回
0 成功（FWK_SUCCESS）；负值为错误码
返回值
FWK_E_DATA内存区域数量超过硬件限制
FWK_E_DEVICE不支持的 CMN-600 修订版本
前置条件
ctx 必须已初始化，且 cmn600_discovery() 已成功执行
后置条件
RN-SAM 寄存器已配置完成并使能（STATUS 寄存器写入 UNSTALL）

根据模块配置中的内存区域映射表，配置指定 RN-SAM 节点的地址映射， 包括 IO 区域、CCIX 区域、系统缓存区域和非哈希系统缓存区域。 配置完成后使能 RN-SAM。
