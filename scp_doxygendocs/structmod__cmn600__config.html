CMN-600 模块配置数据

包含 CMN-600 互连的完整硬件配置信息， 由平台定义并通过 SCP 框架在模块初始化时传入。 该结构体定义了互连 mesh 拓扑、节点映射和时钟依赖等。

◆ hnf_cal_mode

bool mod_cmn600_config::hnf_cal_mode

HN-F CAL 支持使能标志

当设为 true 时，使能 HN-F 的 CAL (Coherent Address Lookup) 模式。 此标志仅当 HN-F 连接到 CAL 端口时生效 （连接到 CAL 端口时，HN-F 的节点 ID 为奇数）。

注解
CAL 模式仅 CMN-600 r2p0 及以上版本支持
