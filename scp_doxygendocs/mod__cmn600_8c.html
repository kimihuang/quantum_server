详细描述

CMN-600 互连控制器模块主实现

实现 CMN-600 模块的 SCP 框架接口和核心功能，包括：

  • 模块初始化 (init)：验证配置参数，定位根节点
  • 模块绑定 (bind)：绑定定时器和系统信息模块 API
  • 模块启动 (start)：执行拓扑发现和 SAM 配置
  • 拓扑发现 (cmn600_discovery)：遍历 mesh 中的所有 XP 和节点
  • 节点配置 (cmn600_configure)：配置 HN-F、RN-SAM 等节点
  • SAM 配置 (cmn600_setup_sam)：配置系统地址映射
  • PPU 观察者回调：外部 RN-SAM 节点上电后重新配置
  • CCIX 配置 API：供 SCMI 模块查询和配置 CCIX
  • 时钟通知处理：等待时钟就绪后执行初始化

static int cmn600_discovery ( void  )
static

扫描 CMN-600 互连拓扑

遍历 mesh 中所有 XP 和其子节点，统计：

  • 外部 RN-SAM 节点数量
  • 内部 RN-SAM 节点数量
  • HN-F 节点数量
  • RN-D、RN-I 节点数量
  • CCIX 相关节点（CXRA、CXHA、CXLA） RN-F 数量通过 RN-SAM 总数减去 RN-D、RN-I 和 CXHA 数量推算。
返回
FWK_SUCCESS 成功；FWK_E_DATA 节点数量超限；FWK_E_RANGE RN-F 数量超限
