详细描述

CCIX Gateway Requesting Agent (CXRA) 寄存器映射

CXRA 是 CCIX 网关中的请求代理，负责向远端 Home Agent 发起一致性请求。 该结构体映射了 CXRA 的所有可编程寄存器，包括节点信息、 LDID-to-RAID 查找表、SAM 地址区域配置和链路控制。
